Politique de Confidentialité

1. Introduction

Bienvenue sur ScientifiBot Y ! Cette politique de confidentialité explique quelles données nous collectons, comment nous les utilisons et quels sont vos droits concernant vos données personnelles.

2. Données Collectées

Lorsque vous utilisez ScientifiBot Y, nous pouvons collecter les informations suivantes :

  • Identifiants Discord (User ID, Server ID, username)
  • Messages et interactions avec le bot
  • Données de logs techniques (commandes utilisées, horodatage)

3. Utilisation des Données

Les données collectées sont utilisées pour :

  • Fournir les fonctionnalités du bot
  • Améliorer les services proposés
  • Garantir la sécurité et prévenir les abus

4. Partage des Données

Nous ne partageons pas vos données personnelles avec des tiers, sauf en cas d'obligation légale.

5. Conservation des Données

Les données sont conservées aussi longtemps que nécessaire pour fournir les services du bot ou pour respecter des obligations légales.

6. Vos Droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : Vous pouvez demander quelles données nous détenons sur vous.
  • Droit de rectification : Vous pouvez demander la correction de données inexactes.
  • Droit à l'effacement : Vous pouvez demander la suppression de vos données personnelles.

Pour exercer vos droits, contactez-nous sur le serveur discord : https://discord.gg/K4H4xhHqUb.

7. Modifications

Nous nous réservons le droit de modifier cette politique de confidentialité. Les modifications seront publiées sur cette page.
